Description

Name - Jonathan Mascarenhas

Course - CSC 468. Section - M01

The custom cat tree creator is a web application that uses HTML, CSS, JavaScript, nodeJS along with a couple of other third party libraries such as formidable, express, mustache, etc. It consists of two main features - the main page and the file management page, both of which are described below.

The Main Page

Buttons

  • Change Component - This rotates an element between House, Pole, and Empty.
  • Add Row - Adds a row of three empty elements.
  • Reset - Changes the content to its default setting of three Empty elements.
  • All Poles - Resets the content and converts the first two rows to all poles.
  • Undo - Undoes the state to its former state.
  • Redo - Redoes the state if an undo previously took place.

    • Below the buttons we have the elements that make up the custom cat tree, and these elements are encompassed by a border that grows and shrinks with the size of the elements inside it. These elements all make up the main content/the tree itself.
